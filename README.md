# 🌿 IndoorPlants — Plant Selling Landing Page

A clean, responsive landing page for an indoor plant e-commerce store. Built as a front-end project and deployed on Vercel.

🔗 **Live Demo:** [https://indooe-plant-selling.vercel.app](https://indooe-plant-selling.vercel.app)

---

## 📸 Overview

IndoorPlants is a visually appealing static website designed to showcase and sell indoor plants online. It highlights key product offerings, brand values, and customer testimonials — all wrapped in a modern, nature-inspired UI.

---

## ✨ Features

- **Hero Section** — Engaging headline with call-to-action buttons ("Shop Now" and "Know More"), including a 30-day free trial badge
- **Value Propositions** — Fast Delivery, Great Customer Service, Original Plants, and Affordable Pricing
- **About Section** — Two-column layout introducing the brand story with plant imagery
- **Popular Plants** — Product cards featuring four plants with scientific names and pricing:
  - 🌿 Boston Fern (*Nephrolepis exaltata*) — $5
  - 🌱 Rubber Plant (*Ficus elastica*) — $5
  - 🌸 Desert Rose (*Adenium obesum*) — $5
  - 🤍 Peace Lily (*Spathiphyllum wallisii*) — $5
- **Customer Reviews** — Testimonial slider with reviewer names and roles
- **Newsletter Subscription** — Email capture form in the footer
- **Footer** — Quick links, popular services, and contact info

## 🚀 Getting Started

### Run Locally
```bash
# Clone the repository
git clone https://github.com/your-username/indooe-plant-selling.git

# Navigate into the project
cd indooe-plant-selling

# Open in browser
open index.html
```

No build tools or dependencies required — plain HTML/CSS/JS.

### Deploy on Vercel
```bash
npm install -g vercel
vercel
```
